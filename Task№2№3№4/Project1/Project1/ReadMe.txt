Задания №2, №3, №4 по OpenMP.
В файле task3.cpp - реализация функций перемножения матрицы на вектор 
и функций перменожения матриц. Перемножение матрицы на вектор - в построчном, постолбцовом и блочном видах.
Перемножение матриц - в неоптимизированном, столбцовом (по B), и в блочном виде. Для каждой вида по три типа 
реализациии параллельности - последовательный, параллельный, параллельный с копированием данных для каждой подзадачи
в новую память.
Результат работы программы - файлы outMV.txt и outMM.txt с измерениями времени работы функций 
в зависимости от количества потоков. 
Для визуализации полученных данных в виде графика используется скрипт graph.py
